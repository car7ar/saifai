# Saif AI - Prompt Engineering Hub

A professional, high-end digital storefront for AI prompts. Built with a focus on premium aesthetics, mobile responsiveness, and cloud synchronization.

## 🚀 Features
- **Premium UI/UX**: Cinematic dark/light modes with glassmorphism effects.
- **Deep Focus Mode**: Immersive full-screen viewing for complex prompts.
- **Admin Dashboard**: Secure management of arts, categories, and custom pages.
- **Cloud Ready**: Prepared for Supabase integration.
- **Mobile Optimized**: Native-app feel with a smart bottom navigation bar.

## 🛠️ Tech Stack
- HTML5 / Vanilla CSS
- JavaScript ES6+
- Lucide Icons
- Supabase (Backend Ready)

## 📦 Deployment
Ready for deployment on **GitHub Pages**, **Vercel**, or **Netlify**. Simply upload the `index.html` file.
